# 法规概述

> 中国无人机管理法规体系架构与核心概念

---

## 一、中国无人机管理法规体系

中国无人机管理法规体系采用**三个层级**的结构，从法律到部门规章形成完整的监管框架。

### 1.1 法规层级架构

```
┌─────────────────────────────────────────────────────┐
│                    第一层：法律                       │
│          《中华人民共和国民用航空法》                   │
│                                                     │
├─────────────────────────────────────────────────────┤
│                   第二层：行政法规                      │
│        《无人驾驶航空器飞行管理暂行条例》                │
│              （国务院、中央军委令第761号）               │
│                                                     │
├─────────────────────────────────────────────────────┤
│                   第三层：部门规章                      │
│        《民用无人驾驶航空器运行安全管理规则》            │
│                    （CCAR-92部）                      │
│                                                     │
│        配套规范性文件：                                │
│        - 《民用无人机驾驶员管理规定》                   │
│        - 《民用无人机空中交通管理办法》                 │
│        - 《民用无人机实名登记管理规定》                 │
│        - 各类咨询通告（AC）、管理文件（MD）              │
└─────────────────────────────────────────────────────┘
```

### 1.2 各层级主要法规

| 层级 | 法规名称 | 发布机关 | 效力说明 |
|------|----------|----------|----------|
| **法律** | 《中华人民共和国民用航空法》 | 全国人大常委会 | 民航领域基本法律，2021年修订加入无人机相关条款 |
| **行政法规** | 《无人驾驶航空器飞行管理暂行条例》 | 国务院、中央军委 | 2024年1月1日起施行，无人机管理专门法规 |
| **部门规章** | 《民用无人驾驶航空器运行安全管理规则》（CCAR-92部） | 中国民航局 | 配套《暂行条例》的部门规章，细化运行管理要求 |
| **规范性文件** | 《民用无人机驾驶员管理规定》（AC-61-FS-2018-20R2） | 中国民航局飞标司 | 规范驾驶员执照管理 |
| **规范性文件** | 《民用无人机空中交通管理办法》（MD-TRS-2018-01） | 中国民航局空管办 | 规范无人机空中交通管理 |
| **规范性文件** | 《民用无人机实名登记管理规定》 | 中国民航局 | 规范无人机实名登记 |

> **法规时效提示**：以上法规截至2024年有效。法规处于持续更新中，请以CAAC官方最新发布为准。

## 二、无人机管理分类

根据《无人驾驶航空器飞行管理暂行条例》，中国按照无人机的**空机重量**和**性能指标**实行分类管理。

### 2.1 无人机分类标准

| 分类 | 空机重量 | 最大起飞重量 | 最大平飞速度 | 主要管理要求 |
|------|----------|--------------|--------------|--------------|
| **微型** | < 0.25kg | < 0.25kg | ≤ 40km/h | 适飞空域飞行无需执照，需实名登记 |
| **轻型** | ≤ 4kg | ≤ 7kg | ≤ 100km/h | 需实名登记，适飞空域无需执照 |
| **小型** | ≤ 15kg | ≤ 25kg | - | 需实名登记，需操控员执照 |
| **中型** | ≤ 116kg | ≤ 150kg | - | 需实名登记，需操控员执照 |
| **大型** | > 116kg | > 150kg | - | 按有人驾驶航空器管理，需完整执照体系 |

### 2.2 各分类核心区别

| 管理维度 | 微型 | 轻型 | 小型 | 中型 | 大型 |
|----------|------|------|------|------|------|
| 实名登记 | 需 | 需 | 需 | 需 | 需 |
| 操控员执照 | 不需 | 不需 | 需 | 需 | 需 |
| 适飞空域 | 可飞行 | 可飞行（限高120m） | 需申请 | 需申请 | 需申请 |
| 责任保险 | 鼓励 | 需投保 | 需投保 | 需投保 | 需投保 |
| 广播式自动发送识别信息 | 不需 | 需 | 需 | 需 | 需 |

> **注意**：以上重量标准依据《无人驾驶航空器飞行管理暂行条例》第七条规定。具体条款请参考CAAC最新法规。

## 三、关键概念解释

### 3.1 VLOS（Visual Line of Sight）

**视距内飞行**，指无人机在操控员或观测员**目视视距范围内**的运行。

| 项目 | 说明 |
|------|------|
| 目视范围 | 通常指半径500米、相对高度低于120米的区域 |
| 要求 | 操控员必须能直接目视无人机，不能依赖FPV或其他辅助设备 |
| 适用场景 | 航拍、测绘、农业植保等日常作业 |
| 执照要求 | 需持有视距内驾驶员执照或超视距驾驶员执照 |

### 3.2 BVLOS（Beyond Visual Line of Sight）

**超视距飞行**，指无人机在操控员或观测员**目视视距范围外**的运行。

| 项目 | 说明 |
|------|------|
| 飞行范围 | 超出半径500米或高度120米的范围 |
| 要求 | 需通过地面站或数据链进行远程操控，通常需要空域审批 |
| 适用场景 | 长距离巡检、物流运输、应急救援等 |
| 执照要求 | 需持有超视距驾驶员执照 |

### 3.3 UOM（Unmanned Aircraft System Operation Management）

**民用无人驾驶航空器综合管理平台**，是CAAC建立的官方管理系统。

| 项目 | 说明 |
|------|------|
| 官方网址 | https://uom.caac.gov.cn/ |
| 功能 | 实名登记、飞行活动申请、操控员执照管理、飞行计划申报 |
| 使用主体 | 无人机所有者、操控员、运营企业 |
| 重要性 | 所有无人机飞行活动的法定申报渠道 |

### 3.4 其他重要术语

| 术语 | 英文全称 | 含义 |
|------|----------|------|
| **适飞空域** | - | 指划设的允许无人机飞行的空域，在此空域内符合条件的无人机可自由飞行 |
| **管制空域** | Controlled Airspace | 需要经过空管部门批准才能进入的空域 |
| **真高** | AGL (Above Ground Level) | 无人机相对于地面表面的实际高度 |
| **海拔高度** | MSL (Mean Sea Level) | 无人机相对于平均海平面的高度 |
| **融合空域** | Integrated Airspace | 无人机与有人驾驶航空器共同运行的空域 |
| **隔离空域** | Segregated Airspace | 专门划设给无人机使用的空域 |
| **地理围栏** | Geofencing | 通过技术手段限制无人机进入特定区域的虚拟边界 |
| **电子围栏** | Electronic Fencing | 无人机内置的禁飞区/限飞区数据系统 |

## 四、法规演进时间线

| 时间 | 事件 | 意义 |
|------|------|------|
| 2013年 | 《民用无人驾驶航空器系统驾驶员管理暂行规定》发布 | 首次建立无人机驾驶员管理制度 |
| 2017年 | 《民用无人机驾驶员管理规定》（AC-61-FS-2016-20R1） | 规范驾驶员执照管理 |
| 2018年 | 《民用无人机驾驶员管理规定》（AC-61-FS-2018-20R2） | 修订驾驶员分类和管理要求 |
| 2021年 | 《中华人民共和国民用航空法》修订 | 法律层面纳入无人机管理 |
| 2023年 | 《无人驾驶航空器飞行管理暂行条例》公布 | 首部无人机专门行政法规 |
| 2024年1月 | 《暂行条例》正式施行 | 中国无人机监管进入新阶段 |
| 2024年 | CCAR-92部配套规章征求意见 | 细化运行管理规则 |
