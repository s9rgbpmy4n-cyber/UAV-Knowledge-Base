# 执照体系

> CAAC无人机驾驶员执照等级、申请条件与考试要求

---

## 一、CAAC无人机驾驶员执照概述

中国民航局（CAAC）对无人机操控员实行**执照管理制度**。根据《民用无人机驾驶员管理规定》（AC-61-FS-2018-20R2）及《无人驾驶航空器飞行管理暂行条例》，操控小型、中型、大型无人机必须持有相应的操控员执照。

### 1.1 执照管理基本原则

| 原则 | 说明 |
|------|------|
| **分类管理** | 按照无人机类别和飞行类型实行分类管理 |
| **执照分级** | 执照分为视距内驾驶员、超视距驾驶员、教员三个等级 |
| **定期更新** | 执照需定期审核更新，确保持证人能力符合要求 |
| **全国统一** | 由民航局飞行标准司统一管理，全国通用 |

---

## 二、执照等级体系

### 2.1 三个执照等级

| 等级 | 名称 | 英文 | 权限说明 |
|------|------|------|----------|
| **等级一** | 视距内驾驶员 | VLOS Pilot | 可在视距范围内操控相应类别的无人机 |
| **等级二** | 超视距驾驶员 | BVLOS Pilot | 可在视距外操控相应类别的无人机，具有申请作业空域的权利 |
| **等级三** | 教员 | Instructor | 具有培训驾驶员的资质，可担任培训机构的教学人员 |

### 2.2 等级递进关系

```
┌──────────────────────────────────────┐
│           教员 (Instructor)           │
│         ↑ 需持超视距执照满1年          │
│         ↑ 通过教员培训和考试            │
├──────────────────────────────────────┤
│        超视距驾驶员 (BVLOS)           │
│         ↑ 需持视距内执照               │
│         ↑ 通过超视距培训与考试          │
├──────────────────────────────────────┤
│        视距内驾驶员 (VLOS)            │
│         ↑ 通过基础培训与考试            │
└──────────────────────────────────────┘
```

> **说明**：教员等级需在取得超视距驾驶员执照后，积累一定飞行经历，并通过教员培训及考试后方可申请。

---

## 三、各等级申请条件

### 3.1 基本条件（所有等级通用）

| 条件 | 具体要求 |
|------|----------|
| **年龄** | 年满16周岁 |
| **文化程度** | 具有初中或以上文化程度 |
| **品行记录** | 无犯罪记录（特别是危害国家安全、公共安全的犯罪记录） |
| **健康状况** | 矫正视力1.0以上，无色盲色弱；无传染性疾病；无精神类疾病 |
| **品行要求** | 五年内无吸食毒品记录，无酗酒滋事记录 |

### 3.2 视距内驾驶员申请条件

| 条件 | 要求 |
|------|------|
| 基本条件 | 满足上述通用条件 |
| 培训要求 | 在具有资质的培训机构完成规定课程培训 |
| 培训学时 | 理论培训不少于 XX 学时（具体条款请参考CAAC最新法规） |
| 飞行训练 | 完成规定时间的飞行训练 |

### 3.3 超视距驾驶员申请条件

| 条件 | 要求 |
|------|------|
| 前置条件 | 已取得视距内驾驶员执照 |
| 培训要求 | 在具有资质的培训机构完成超视距课程培训 |
| 飞行经历 | 具有一定的视距内飞行经历记录 |
| 地面站操作 | 完成地面站操作培训 |

### 3.4 教员申请条件

| 条件 | 要求 |
|------|------|
| 前置条件 | 已取得超视距驾驶员执照 |
| 执照持有时间 | 持有超视距驾驶员执照满 **1年** 以上 |
| 飞行经历 | 具有一定的飞行教学辅助经历 |
| 培训要求 | 完成教员培训课程并通过考试 |

## 四、理论考试内容

### 4.1 考试科目

| 科目 | 内容范围 | 考试形式 |
|------|----------|----------|
| **航空法规** | 民用航空法、无人机管理暂行条例、相关规章 | 计算机考试，选择题 |
| **航空器一般知识** | 无人机结构原理、动力系统、飞控系统 | 计算机考试，选择题 |
| **气象学** | 基本气象知识、风、能见度、天气现象对飞行的影响 | 计算机考试，选择题 |
| **空气动力学基础** | 升力、阻力、飞行基本原理 | 计算机考试，选择题 |
| **飞行性能与计划** | 航程计算、性能限制、飞行计划制定 | 计算机考试，选择题 |
| **人为因素** | 飞行决策、压力管理、差错管理 | 计算机考试，选择题 |
| **运行程序** | 飞行前检查、正常程序、应急程序 | 计算机考试，选择题 |
| **通信** | 无线电通信基础、标准术语 | 计算机考试，选择题 |

### 4.2 各等级理论考试要求

| 等级 | 合格分数线 | 备注 |
|------|------------|------|
| 视距内驾驶员 | 70分 | （具体条款请参考CAAC最新法规） |
| 超视距驾驶员 | 80分 | （具体条款请参考CAAC最新法规） |
| 教员 | 80分 | （具体条款请参考CAAC最新法规） |

## 五、实操考试内容

### 5.1 视距内驾驶员实操考试

| 考试项目 | 内容说明 |
|----------|----------|
| **飞行前检查** | 对无人机进行完整的飞行前检查 |
| **起飞** | 按照程序完成起飞动作 |
| **悬停** | 在指定位置保持悬停 |
| **慢速水平位移** | 按要求进行前后左右缓慢移动 |
| **水平8字飞行** | 完成水平方向的8字航线飞行 |
| **定点降落** | 在指定区域精确降落 |
| **应急操作** | 模拟应急情况下的正确处置 |

### 5.2 超视距驾驶员实操考试

| 考试项目 | 内容说明 |
|----------|----------|
| **基础飞行** | 包含视距内驾驶员所有考试项目 |
| **地面站操作** | 使用地面控制站进行航线规划与自主飞行 |
| **应急返航** | 模拟通信中断等应急情况的返航操作 |
| **应急处置** | 更复杂的应急处置场景考核 |

### 5.3 教员实操考试

| 考试项目 | 内容说明 |
|----------|----------|
| **教学演示** | 对标准飞行动作进行教学演示 |
| **教学讲解** | 对理论知识点进行讲解演示 |
| **应急教学** | 演示应急处置的教学方法 |

## 六、执照类别与签注

### 6.1 无人机类别

执照按照无人机的类别进行签注，主要包括：

| 类别 | 说明 |
|------|------|
| **多旋翼** | 多旋翼无人机（最常见类别） |
| **固定翼** | 固定翼无人机 |
| **直升机** | 无人直升机 |
| **垂直起降固定翼** | VTOL固定翼无人机 |

### 6.2 执照有效期与续期

| 项目 | 说明 |
|------|------|
| **有效期** | 执照有效期为 **6年**（具体条款请参考CAAC最新法规） |
| **续期要求** | 需在到期前申请续期，提供飞行经历记录 |
| **飞行经历要求** | 续期需满足一定的飞行经历时间要求 |
| **逾期处理** | 逾期需重新参加部分考试（具体条款请参考CAAC最新法规） |
| **定期检查** | 在有效期内需完成定期检查（具体条款请参考CAAC最新法规） |

## 七、ASFC执照与CAAC执照的区别

### 7.1 两个体系的对比

| 对比维度 | CAAC执照 | ASFC执照 |
|----------|----------|----------|
| **全称** | 民用无人驾驶航空器操控员执照 | 遥控航空模型飞行员执照 |
| **发证机关** | 中国民用航空局（CAAC） | 国家体育总局航空无线电模型运动管理中心（ASFC） |
| **管理范围** | 民用无人机飞行活动 | 航空模型运动 |
| **法律效力** | 具有法定强制性，商业飞行必须持有 | 主要用于航模运动和赛事 |
| **适用场景** | 商业飞行、行业应用、专业作业 | 航模爱好、竞技比赛 |
| **适用重量** | 小型及以上无人机（>7kg）必须 | 主要适用于航模（通常<25kg） |
| **国际互认** | 部分国家/地区认可 | 主要在国内航模圈使用 |
| **培训体系** | 民航局认可的培训机构 | 航模协会认可的培训机构 |

### 7.2 如何选择

| 使用目的 | 建议执照 |
|----------|----------|
| 商业航拍、测绘、巡检等 | **CAAC执照** |
| 农业植保作业 | **CAAC执照** |
| 无人机物流配送 | **CAAC执照** |
| 航模爱好、竞技比赛 | ASFC执照 |
| 个人娱乐飞行（轻型以下） | 通常无需执照（在适飞空域内） |

> **重要提示**：进行任何商业性质的无人机飞行活动，**必须持有CAAC颁发的操控员执照**。ASFC执照不能替代CAAC执照用于商业飞行。

## 八、执照申请流程

### 8.1 申请步骤

```
步骤1：选择培训机构
    ↓ 选择民航局认可的无人机培训机构报名
步骤2：参加理论培训
    ↓ 完成规定学时的理论课程学习
步骤3：参加飞行训练
    ↓ 在教练指导下完成规定时间的飞行训练
步骤4：理论考试
    ↓ 通过培训机构向民航局申请理论考试
步骤5：实操考试
    ↓ 理论通过后申请实操考试
步骤6：申请执照
    ↓ 理论和实操均通过后，在UOM平台提交执照申请
步骤7：领取执照
    ↓ 民航局审核通过后发放电子/实体执照
```

### 8.2 培训机构资质

- 培训机构需取得民航局颁发的**无人机驾驶员训练机构合格证**
- 可在UOM平台查询全国已批准的培训机构名单
- 选择培训机构时注意确认其培训类别（多旋翼/固定翼等）是否匹配需求
