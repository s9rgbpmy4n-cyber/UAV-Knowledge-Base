# 安全规范

> 无人机飞行安全操作规范与应急处置

---

## 一、飞行前检查清单

### 1.1 设备检查

| 检查项 | 检查内容 | 法规依据 |
|--------|----------|----------|
| 机体检查 | 机臂、机身是否有裂纹或损伤，螺丝紧固 | CCAR-92部运行要求 |
| 螺旋桨 | 桨叶完好、紧固、无变形、无裂痕 | 飞行前强制检查 |
| 电机 | 电机转动顺畅、无异响、无松动 | 设备适航要求 |
| 电池 | 电量充足（≥30%余量）、无鼓包、接口清洁 | 锂电池安全管理规范 |
| 飞控系统 | 上电自检通过、GPS信号≥8颗星 | 导航定位要求 |
| 遥控器 | 电量充足、信号连接正常、失控保护已设 | 遥控链路要求 |
| 图传/数传 | 图像清晰、数据链路稳定 | 通信链路要求 |
| 存储卡 | 空间充足、读写正常 | 数据记录要求 |

### 1.2 法规符合性检查

| 检查项 | 检查内容 | 法规依据 |
|--------|----------|----------|
| 执照有效性 | 操控员执照在有效期内 | CCAR-61部 |
| 实名登记 | 无人机已在UOM平台完成实名登记 | 实名制登记管理规定 |
| 保险 | 已购买有效第三者责任险（≥10万元） | 运行管理要求 |
| 空域申请 | 已在UOM平台完成飞行活动申请 | CCAR-92部 |
| 禁飞区确认 | 飞行区域不在机场净空保护区、军事管理区等国家规定的禁飞空域内 | 《无人驾驶航空器飞行管理暂行条例》 |

## 二、气象条件要求

### 2.1 一般气象限制

| 气象要素 | 限制标准 | 说明 |
|----------|----------|------|
| 风速 | ≤6级（≤12m/s） | 超过需评估机型抗风能力 |
| 能见度 | ≥500m（VLOS） | 保证操控员目视观察 |
| 温度 | -10°C ~ 45°C | 注意低温对电池性能影响 |
| 降水 | 禁止在雨、雪、冰雹中飞行 | 防水机型除外 |
| 云底高 | ≥150m（农业平原）/ ≥300m（山区） | 保证安全间隔 |

### 2.2 禁止飞行的气象条件

- 雷暴、闪电、强风切变等危险天气
- 沙尘暴、大雾等能见度极低天气
- 气温低于电池安全工作温度

## 三、电池安全规范

### 3.1 锂电池使用安全

| 环节 | 安全要求 |
|------|----------|
| 充电 | 使用原装充电器，充电时有人看管，远离易燃物，充电环境通风 |
| 存储 | 半电量（3.8-3.85V/节）存储，存放于阴凉干燥处，温度15-25°C |
| 运输 | 遵守《锂电池航空运输规范》，做好绝缘防护 |
| 携带 | 民航旅客行李中锂电池额定能量超过160Wh严禁携带 |
| 报废 | 电量放空后按电子废弃物规范处理，禁止随意丢弃 |

### 3.2 电池检查要点

- 电池外观无鼓包、无漏液、无破损
- 接口清洁、无腐蚀
- 电量满足飞行任务+30%备用余量
- 6S电池有7根引出线（粗红黑2根输出+细5根平衡线）

## 四、飞行中安全注意事项

### 4.1 飞行安全原则

1. 始终保持对无人机的直接目视接触（VLOS），操控员与无人机距离≤500m
2. 遵守批准的飞行计划，不擅自扩大飞行范围
3. 飞行真高一般≤120m，确需超高度应提前申请
4. 注意周围环境，与人群、建筑物、高压线保持安全距离
5. 随时准备执行应急程序，确保地面人员和财产安全

### 4.2 避让规则

| 情况 | 处置要求 |
|------|----------|
| 遇到有人驾驶航空器 | 无人机必须避让，立即下降高度或返航 |
| 接近障碍物 | 保持安全间隔，必要时悬停评估 |
| 地面人员接近 | 悬停警告，必要时降落 |

### 4.3 飞行高度限制

| 区域类型 | 高度限制 |
|----------|----------|
| 一般区域 | 真高≤120m |
| 机场净空保护区 | 禁止飞行（除非特别批准） |
| 人口密集区 | 需特别评估，一般限制更严格 |

## 五、应急处置流程

### 5.1 常见应急情况及处置

| 应急情况 | 处置要点 |
|----------|----------|
| 失去GPS信号 | 切换姿态模式，保持悬停，尝试恢复信号，必要时手动返航 |
| 通信中断 | 触发失控保护（自动返航或原地降落），立即寻找降落点 |
| 电量不足 | 立即执行返航程序，电量低于20%就近降落，禁止继续飞行 |
| 遇到其他航空器 | 立即下降高度避让，必要时降落 |
| 恶劣天气突变 | 立即中断任务，优先返航或就近降落 |
| 设备故障 | 判断故障等级，轻微故障可控返航，严重故障立即迫降 |

### 5.2 应急处置原则

1. **人员安全为最高优先级** — 任何情况下以保护地面人员安全为第一
2. **减少财产损失** — 在确保人员安全前提下，尽量减少财产损失
3. **及时报告** — 发生不安全事件后及时向相关部门报告
4. **保留记录** — 保存飞行数据记录作为调查依据

## 六、事故报告要求

### 6.1 报告义务

| 情况 | 报告时限 | 报告对象 |
|------|----------|----------|
| 造成人员伤亡 | 立即报告 | 民航地区管理局、公安机关 |
| 造成重大财产损失 | 24小时内 | 民航地区管理局 |
| 空中相撞 | 立即报告 | 民航地区管理局、空中交通管制单位 |
| 其他不安全事件 | 48小时内 | 民航地区管理局 |

### 6.2 报告内容

- 事件发生时间、地点
- 无人机型号、登记编号
- 操控员信息
- 事件经过描述
- 损失情况
- 初步原因分析

## 七、数据记录与保存

### 7.1 飞行数据记录要求

| 数据类型 | 记录内容 | 保存期限 |
|----------|----------|----------|
| 飞行日志 | 起飞时间、降落时间、飞行轨迹 | ≥12个月 |
| 设备状态 | 电池状态、GPS信号、飞行参数 | ≥12个月 |
| 操控员信息 | 执照编号、飞行时间 | 长期保存 |
