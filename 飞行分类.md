# 飞行分类

> CAAC无人机飞行类型分类、执照要求与空域限制

---

## 一、视距内飞行（VLOS）

### 1.1 定义

视距内飞行（Visual Line of Sight, VLOS），是指无人机在**操控员或观测员目视视距范围内**的飞行活动。

### 1.2 核心要求

| 要求项 | 具体标准 |
|--------|----------|
| **水平距离** | 无人机与操控员之间的水平距离不超过 **500米** |
| **垂直高度** | 无人机相对地面高度不超过 **120米（真高）** |
| **目视条件** | 操控员必须能够**直接目视**观察无人机，不能使用FPV眼镜或其他替代视觉设备作为主观察手段 |
| **气象条件** | 能见度良好，昼间飞行（日出至日落） |

### 1.3 操控员执照要求

| 无人机类别 | 执照要求 |
|------------|----------|
| 微型无人机 | 不需要执照，但需了解相关法规 |
| 轻型无人机（适飞空域内） | 不需要执照 |
| 轻型无人机（管制空域内） | 需要视距内驾驶员执照 |
| 小型及以上无人机 | 需要视距内或超视距驾驶员执照 |

### 1.4 适用场景

- 航拍、摄影
- 农业植保（小面积）
- 短距离巡检
- 测绘（小范围）
-  recreational 娱乐飞行

---

## 二、超视距飞行（BVLOS）

### 2.1 定义

超视距飞行（Beyond Visual Line of Sight, BVLOS），是指无人机在**操控员或观测员目视视距范围以外**的飞行活动。

### 2.2 核心要求

| 要求项 | 具体标准 |
|--------|----------|
| **飞行范围** | 超出水平500米或高度120米的限制 |
| **操控方式** | 通过地面控制站（GCS）和数据链路进行远程操控 |
| **通信链路** | 必须具备可靠的通信链路，确保全程可控 |
| **感知与避让** | 需具备或配合相应的感知与避让（DAA）能力 |
| **空域审批** | 必须提前向空管部门申请并获得批准 |

### 2.3 操控员执照要求

| 要求 | 说明 |
|------|------|
| **执照等级** | 必须持有**超视距驾驶员执照** |
| **飞行经验** | 通常要求先取得视距内执照并积累一定飞行小时数 |
| **地面站操作** | 需通过地面站操作考核 |
| **理论水平** | 需掌握空域管理、气象学、通信链路等相关知识 |

### 2.4 适用场景

- 长距离电力/管道巡检
- 无人机物流配送
- 大面积测绘与地理信息采集
- 应急救援与搜救
- 通信中继

## 三、不同飞行类型的执照要求对照

| 飞行类型 | 微型 | 轻型 | 小型 | 中型 | 大型 |
|----------|------|------|------|------|------|
| 视距内飞行（适飞空域） | 无需执照 | 无需执照 | 需视距内/超视距执照 | 需视距内/超视距执照 | 不适用 |
| 视距内飞行（管制空域） | 需视距内执照 | 需视距内执照 | 需视距内/超视距执照 | 需视距内/超视距执照 | 不适用 |
| 超视距飞行 | 需超视距执照 | 需超视距执照 | 需超视距执照 | 需超视距执照 | 需超视距执照 |

> **说明**：具体条款请参考CAAC最新法规，不同地区可能有细化要求。

## 四、飞行高度限制

### 4.1 一般高度限制

| 场景 | 高度限制 | 说明 |
|------|----------|------|
| **适飞空域（轻型）** | ≤ 120米 真高 | 无需申请，直接飞行 |
| **适飞空域（小型及以上）** | ≤ 120米 真高 | 需提前在UOM平台报备 |
| **管制空域** | 按审批高度 | 需经空管部门批准 |
| **机场净空保护区** | 严格禁飞 | 以机场跑道中心线为准的椭圆形区域 |

### 4.2 特殊高度审批

如需在120米以上高度飞行，需满足以下条件：

1. 通过UOM平台提交飞行计划申请
2. 说明飞行高度、范围、时间的必要性
3. 获得空管部门的书面批准
4. 配备必要的通信和监视设备

## 五、禁飞区与限飞区

### 5.1 禁飞区（绝对禁止飞行）

| 区域类型 | 范围 | 法律依据 |
|----------|------|----------|
| **机场净空保护区** | 民用机场跑道中心线两侧各10公里、跑道端外20公里范围内的区域 | 暂行条例第十九条 |
| **军事管理区** | 军事禁区、军事管理区以及周边规定范围 | 暂行条例第十九条 |
| **重要政治区域** | 国家机关、重要军事设施周边 | 暂行条例第十九条 |
| **核设施区域** | 核电站、核设施周边规定范围 | 暂行条例第十九条 |
| **大型活动区域** | 依法划设的大型群众性活动、重要赛事等临时禁飞区域 | 暂行条例第十九条 |
| **其他法定区域** | 法律、行政法规规定的其他禁飞区域 | 暂行条例第十九条 |

### 5.2 限飞区（限制条件飞行）

| 区域类型 | 限制条件 | 说明 |
|----------|----------|------|
| **人口密集区** | 需额外安全措施 | 城市中心、商业区等 |
| **重要设施周边** | 需经管理单位同意 | 发电厂、水坝、油库等 |
| **边境地区** | 需特殊审批 | 靠近国界的区域 |
| **临时管控区** | 按临时通告执行 | 重大活动、突发事件期间 |

### 5.3 查询方式

1. **UOM平台**（https://uom.caac.gov.cn/）提供全国空域信息查询
2. ** manufacturer's App**：主流无人机厂商的APP内置电子围栏数据
3. **地方通告**：关注地方民航管理部门发布的临时管控通告

## 六、特殊飞行类型

### 6.1 夜间飞行

| 项目 | 要求 |
|------|------|
| 定义 | 日落之后至日出之前的飞行 |
| 一般规定 | 一般情况下禁止无人机夜间飞行 |
| 例外情况 | 经特别批准的应急救援、巡检等任务 |
| 额外要求 | 需配备防撞灯、照明设备等，并获空管部门特别许可 |

### 6.2 集群飞行

| 项目 | 要求 |
|------|------|
| 定义 | 多架无人机在同一区域协同飞行 |
| 一般规定 | 需提前申请，制定专门的运行方案 |
| 安全要求 | 需具备防撞、协同控制等能力 |
| 表演飞行 | 需取得相关文化、公安、民航等多部门许可 |

### 6.3 水上飞行

| 项目 | 要求 |
|------|------|
| 定义 | 在水面或水面附近区域的飞行 |
| 要求 | 需考虑水上气象、通信链路、回收方案等 |
| 执照要求 | 与普通飞行一致，但需具备水上操作相关培训 |

## 七、飞行前检查清单

| 序号 | 检查项 | 状态 |
|------|--------|------|
| 1 | 确认飞行区域不在禁飞区内 | ☐ |
| 2 | 在UOM平台完成飞行活动申请（如需要） | ☐ |
| 3 | 检查无人机状态（电机、桨叶、电池电量） | ☐ |
| 4 | 确认操控员执照在有效期内 | ☐ |
| 5 | 检查气象条件（风速、能见度、降水） | ☐ |
| 6 | 确认地面站/遥控设备电量充足 | ☐ |
| 7 | 确认GPS信号良好 | ☐ |
| 8 | 购买有效责任保险（小型及以上必须） | ☐ |
| 9 | 了解应急处置程序 | ☐ |
| 10 | 确保现场有安全警戒措施 | ☐ |
